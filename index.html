HTML - One of the Big 3 Technologies

HTML stands fot Hyper Text Markup Language. It is used to create the structure of any web page, website and web application.

HTML Boiler plate includes the following tags:

  • Doctype declares that whatever we are going to code are html5
  • html - every tag goes inside the HTML tag
  • head - Head contains all meta tags and title tags
  • title - title holds the title of the website
  • body - all tags and contents are written inside the BODY tags and all contents are displayed
